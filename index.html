프로필 설정하기

1. 닌텐도 아이디 확인
마이페이지 또는 메인 좌측 상단에서 확인할 수 있습니다.
2. 닌텐도 아이디 작성
확인한 닌텐도 아이디는 프로필 사진란에 적어주세요!
3. 친구코드 확인
통신 시 필요한 친구코드 12자리를 확인해주세요.
4. 주민대표 확인
모동숲 내에서 주민대표 이름을 확인해주세요.
5. 주민대표/친구코드 작성
주민대표와 친구코드는 프로필 이름란에 적어주세요!
6. (선택사항)오픈프로필
오픈프로필로 설정하실 경우, 프로필 사진에는 닌텐도 아이디가 들어간 사진을 첨부해도 괜찮습니다.
